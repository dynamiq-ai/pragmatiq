pragmatiq is an open-source project by [Dynamiq](https://getdynamiq.ai) — a developer-friendly
implementation of the ideas in the PRAGMA paper, built so ML engineers and data scientists can
run, inspect, adapt, and benchmark them on their own data.

## Attribution

> pragmatiq is an independent implementation inspired by the PRAGMA paper
> ([arXiv:2604.08649](https://arxiv.org/abs/2604.08649)) and is not affiliated with or endorsed
> by Revolut.

The project assets are original. They do not include Revolut logos, trade dress, or figures
copied from the paper. PRAGMA and Revolut are referenced only for scholarly attribution.

## Cite

If pragmatiq is useful in your work, please cite the original paper (arXiv:2604.08649) and
link to this repository. See
[`CITATION`](https://github.com/dynamiq-ai/pragmatiq) in the repo for the exact metadata.
